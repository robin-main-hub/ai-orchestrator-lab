우리 코딩 워크벤치(docs/42)와 갭 분석.

3. 보조: Agentic UX & Design Patterns (Sandhya Hegde), AI Chat Interface
   Playbook (follow-up chips, @-mention 패턴).

## 백로그 (우선순위순)

1. **자동 컴팩션**: usage 합계가 세션 설정 한도(기본 64k)의 90% 도달 시
   compactSession 자동 호출 + "압축됨" 시스템 카드 표시.
2. **턴 스냅샷**: 빌드 모드 턴 시작 시 `git stash create` 류 스냅샷을 게이트로
   기록, 실패 턴 후 "파일 되돌리기" 버튼.
3. **패턴 영속 승인**: 승인 다이얼로그에 "이 접두어 세션 동안 허용" — 기존
   extraSafePrefixes에 런타임 추가.
4. **LSP/진단 도구**: edit/write 후 `tsc --noEmit` 자동 제안 카드.
5. **모델 역할 분리**: plan 모드는 저가 모델 프로필로 자동 스위치 옵션.
6. **워크벤치 내 서브에이전트**: `{"tool":"delegate","goal"}` → 병렬 미션 1개
   스핀업 후 결과 회수.

## 함께 고친 것 (이 PR)

- **대화탭 추천대화**: 사용자의 원 지시("에이전트가 해준 답변을 근거로 3개
  생성")대로 재작성 — 세 슬롯(파고들기/실행/검증) 전부 마지막 답변에서 파생,
  생각/응답 중에는 숨김. 답변을 신호(오류·제안 행동·열린 질문·주제)로 분해하는
  순수 분석이라 공급자가 죽어 있어도 동작.
- **코딩 러너 리마인더**: 도구 3라운드째부터 원래 요청을 재주입해 장기 루프
  표류 방지.
